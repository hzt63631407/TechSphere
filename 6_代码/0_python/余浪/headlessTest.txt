2、无头浏览器的测试
2.1、安装使用：pip或npm (先按2.2 解压测试脚本文件 webDeid.tgz 后，进入各自目录安装)
nightmare:
    npm install nightmare
    node nightmareTest.js
phantomjs:
    https://phantomjs.org/download.html
    ./phantomjs/bin/phantomjs.exe phantomjsTest.js
playwrigh:
    npm install playwright
    node playwrighTest.js
puppeteer:
    npm install puppeteer
    node puppeteerTest.js
selnium:
    pip3 install selenium
    python3 seleniumTest.py
splash:
    docker pull scrapinghub/splash
    docker run -d --name=splash -p 8050:8050 scrapinghub/splash
    浏览器打开：http://127.0.0.1:8050
    复制 splashTest.txt 执行
zombie:
    npm install zombie
    node zombieTest.js
