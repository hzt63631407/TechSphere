（3）.本地创建git仓库（不需要）
git克隆基本上是一个组合：
> git init(创建本地存储库)
> git remote add(将URL添加到该存储库)
git fetch(从该URL中获取所有分支到本地存储库)
git checkout(创建工作树中主分支的所有文件)
因此，不，您不必执行git init，因为它已经由git clone完成。


（4）.将git中的工程克隆到本地
在刚创建的本地git仓库中，使用git clone 命令，将git中的工程克隆到本地
